
En este folder se deben agregar los sgte inputs necesarios para la ejecucion:

- carpeta con archivos .csv de condiciones iniciales en funcion de la P.
- .md del configfile editable
- .csv credenciales a las base de datos
- .csv con metadatos que contenga: asignacion de tramos-estaciones y parámetros de curvas de estimación de caudales.
- .csv con asignacion de tramos importantes pero sin estaciones.
- .csv con asignacion de cuencas dentro del simubasin
- .nc con simubasin de ejecucion.
- .json con configuracion del modelo.
